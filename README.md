# YouCode 2026 Hackathon
April 2026
made by: Arielle, Lana, Vivian

<NAME> is an accessible and user-friendly website made for nonprofit organizations to manage volunteers and positions for events and community initiatives with a simple drag and drop. It includes functionality to add volunteers based on availability, skills, and contact info, as well as define volunteer positions based on schedules, volunteer capacity, and required skills. Other features include a "view all" functionality, which allows an overview of all positions and volunteers, in addition to an auto-assign feature that automatically assigns volunteers to positions based on skills and availability. This project uses HTML, CSS, and JavaScript. In the future, we hope to implement a backend to track user accounts and save volunteer data.
